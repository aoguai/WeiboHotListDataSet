# WeiboHotListDataSet

## 微博热榜数据集

抓取了微博热榜 2022-11-25 ~ 2023-03-08 中上榜的博文（其中博文只抓取了上榜当天的博文）与对应的评论数据

## 数据结构

WeiboHotListDataSet.zip

- archives.tar.gz
  
  - 2022-11-25 ~ 2023-03-08 的全部微博热榜词条 ，以 每天、markDown 文件的形式保存

- comments.tar.gz
  
  - 对应的微博热榜词条的博文和与之对应的评论 Excel 文件
